The outcome

A complete, crafted site — fast, and for nothing.

Speed Shipped in days, not weeks

Directed in plain language instead of hand-coded line by line — from an empty repo to a live site in a handful of focused sessions.

Cost Zero budget to build or run

No framework, no licences, no paid tools — plain HTML, CSS & JS, hosted for free on GitHub Pages.

Control Designer-led, end to end

No engineering handoff and no spec documents — I owned every decision, from the design system to the final deploy.

Longevity Built to keep growing

Data-driven by design, so adding a new project or AI experiment is a single entry — never a rebuild.

What's next

A living project, still being built.

This is version one, not a finished artefact. Next on the roadmap:

Planned Google Analytics

Measure how visitors actually move through the work — and treat the portfolio itself as a product I can read the data on and keep improving.

Planned Payload CMS

Move case studies and AI Lab entries into a headless CMS, so new work can be published and edited without touching the code.

AI didn't replace the design work — it collapsed the distance between an idea and a working version of it.

The scarce resource stopped being production time and became judgment. That's the shift I'm betting my practice on — and this is the first proof.
